# Citygates Haulage & Logistics

Company profile website for **Citygates Haulage & Logistics (CHL)** — Lagos, Nigeria.

**Tagline:** Driven by People. Delivered with Precision.

## Tech stack

- Next.js 16 (App Router, Turbopack)
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React icons

## Pages

| Route | Description |
| ----- | ----------- |
| `/` | Company profile homepage |
| `/city-access` | Executive charter & corporate transport |
| `/city-move` | Haulage & cargo (8-tonne fleet) |
| `/cityrun` | Eco-friendly electric bicycle dispatch |

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Images

Photography is stored in `public/images/` (sourced from Unsplash for development). Replace with your own fleet, team, and operations photos for production.
